# @tuturuuu/ai

AI utilities and helpers for Tuturuuu Platform.

## Installation

```bash
npm install @tuturuuu/ai
# or
yarn add @tuturuuu/ai
# or
pnpm add @tuturuuu/ai
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test
```
