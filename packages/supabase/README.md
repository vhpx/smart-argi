# @tuturuuu/supabase

Supabase client and utilities for Tuturuuu Platform.

## Installation

```bash
npm install @tuturuuu/supabase
# or
yarn add @tuturuuu/supabase
# or
pnpm add @tuturuuu/supabase
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test
```
